## ⚡ Getting Started
1. Clone the repository:
``` bash
git clone https://github.com/DawnFallz/dawnfallz.github.io.git

```

2. Go to the project directory:
``` bash
cd dawnfallz.github.io

```

3. Start the development server:
``` bash
npx serve

```
